# Network Segmentation & Firewall Design Lab (Cisco Packet Tracer)

## Objective
Design a secure enterprise network with segmentation and access control.

## Lab Setup
- Cisco Packet Tracer
- pfSense Firewall
- VLAN-based architecture

## Network Design
- Segmented network into:
  - Internal Network
  - DMZ
  - SIEM Network
  - RADIUS Network
  - IDS Network

## Security Controls
- Firewall rules (deny by default)
- VLAN segmentation
- Access Control Lists (ACLs)
- IDS integration (Security Onion)

## Key Implementations
- Restricted communication between network segments
- Protected critical assets (servers & databases)
- Implemented least privilege access

## Results
- Reduced attack surface using segmentation
- Prevented unauthorized lateral movement
- Designed enterprise-style secure architecture

## Tools Used
- Cisco Packet Tracer
- pfSense
- Security Onion

## Skills Demonstrated
- Network Security
- Firewall Configuration
- Network Segmentation
